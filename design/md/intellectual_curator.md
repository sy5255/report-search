# 'Intellectual Curator' 시스템 디자인 계획

사용자의 질문 의도를 파악하고 RAG(문서 검색)와 SQL(DB 분석)을 적재적소에 활용하여 지식을 큐레이션하는 고도화된 지식 검색 플랫폼입니다.

## 1. 핵심 가치 및 컨셉
- **지적 큐레이션 (Intellectual Curation):** 단순 정보 나열이 아닌, 맥락에 맞는 깊이 있는 답변 제공.
- **투명성 및 신뢰성:** 답변의 근거가 되는 원본 문서(RAG)와 쿼리 결과(SQL)를 명확히 제시.
- **데이터 활용의 극대화:** 3년간 축적된 이메일 기반 마크다운 문서를 체계적으로 탐색하고 시각화.

## 2. 주요 화면 구성 및 기능

### ① 메인 채팅 인터페이스 (AI Search & Reasoning)
- **멀티모달 답변창:** 일반 텍스트 답변, SQL 실행 결과(표/간단한 차트), Tool 호출 로그(Thought Process) 표시.
- **RAG 사이드바 (Context Panel):** 답변 생성에 참고된 상위 5개 마크다운 문서 노출. 유사도 점수 및 핵심 스니펫 포함.
- **인터랙티브 요소:** 답변 내의 특정 데이터나 문서를 클릭하면 상세 뷰로 즉시 연결.

### ② 데이터셋 게시판 (Knowledge Repository)
- **전체 문서 아카이브:** 3년 치 이메일 기반 마크다운 문서 리스트.
- **하이브리드 검색:** LLM 기반 검색 외에 익숙한 키워드 검색바 및 날짜/카테고리 필터 제공.
- **문서 상세 뷰:** 마크다운 렌더링, 메타데이터(발신자, 일시) 표시, 관련 문서 추천.

### ③ AI 분석 대시보드 (Dynamic Data Visualization) - *New*
- **자연어 기반 차트 생성:** "지난 3개월간 프로젝트 관련 이메일 빈도를 그래프로 그려줘"와 같은 요청 처리.
- **SQL-to-Chart 워크플로우:** 유저 질문 → LLM SQL 생성 → DB 데이터 추출 → 유저가 선택한 유형(Bar, Line, Pie 등)으로 시각화.
- **위젯 관리:** 생성된 그래프를 대시보드에 고정하거나 리포트 형태로 내보내는 기능.

### ④ 큐레이션 대시보드 (Knowledge Insights)
- **지식 그래프:** 문서 간의 연관 관계를 노드와 링크로 시각화.
- **토픽 클러스터링:** 전체 데이터셋의 주요 주제를 한눈에 파악할 수 있는 워드 클라우드 또는 클러스터 맵.

## 3. 디자인 방향성
- **Clean & Professional:** 방대한 데이터를 다루므로 화이트/라이트 그레이 톤의 깔끔한 UI 지향.
- **고대비 가독성:** 마크다운 문서의 가독성을 위해 최적화된 타이포그래피 적용.
- **직관적인 상태 표시:** AI가 현재 어떤 Tool을 사용 중인지(Searching, Querying, Visualizing) 애니메이션으로 표현.
